## Codes
- Generate env: 
    `conda create env -n flask`
    `conda activate flask`
    `conda install alchemy flask flask_sqlalchemy`
    `conda install -c conda-forge flask-sqlalchemy`
- Generate db file: 
    `from server import db`
    `db.create_all()`
    `exit()`
